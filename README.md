## Usage

1. Download the desired theme:
    - `rose-pine.css`
    - `rose-pine-moon.css`
2. Open Super Productivity
3. Go to **Settings > Theme (experimental) > Install theme...**
4. Select the theme file or files to install
5. Use the theme dropdown to select the installed theme
6. Use Super Productivity’s light/dark mode selector to switch between variants

## Themes

### Rosé Pine

Uses:

- **Dark mode:** Rosé Pine
- **Light mode:** Rosé Pine Dawn

### Rosé Pine Moon

Uses:

- **Dark mode:** Rosé Pine Moon
- **Light mode:** Rosé Pine Dawn

## Notes

Super Productivity’s custom themes follow the app’s current light/dark mode.

These themes include both a dark palette and a light palette in the same file. They do not force Super Productivity into light or dark mode; they follow the app’s light/dark mode selector.

## Known issues

### Theme names may appear with “delete”

After uploading a theme, Super Productivity may display the theme name as something like `Rose Pinedelete`.

This appears to be a Super Productivity UI bug. The word `delete` comes from the delete icon/button rendered next to the theme title, but it is visually merged into the theme name.

The theme itself should still work correctly.

### Theme support is experimental

These themes use Super Productivity’s experimental theme upload feature. Super Productivity’s theme system may change over time, so minor visual issues may appear after Super Productivity updates.
